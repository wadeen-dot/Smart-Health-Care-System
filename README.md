# Smart-Health-Care-System
Smart Healthcare Management System (FYP-I Completed, FYP-II In Progress) developed using HTML, CSS, JavaScript, PHP, and MySQL.

## Final Year Project (FYP)

### Project Status

* FYP-I Completed
* FYP-II In Progress

### Project Description

Smart Healthcare Management System is a web-based application designed to improve healthcare service management and patient record handling. The system aims to provide an efficient and user-friendly platform for managing healthcare-related information.

### Objectives

* Improve healthcare information management
* Organize patient records efficiently
* Enhance accessibility of healthcare services
* Provide a user-friendly web interface

### Features

* Patient Record Management
* Data Management System
* User-Friendly Interface
* Database Integration

 Technologies

* HTML
* CSS
* JavaScript
* PHP
* MySQL
